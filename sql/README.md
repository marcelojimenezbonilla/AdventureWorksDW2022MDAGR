# Scripts SQL

Esta carpeta contiene los scripts SQL para el Data Warehouse.

## Estructura

- **dimensions/**: Scripts para crear y mantener tablas de dimensiones
- **facts/**: Scripts para crear y mantener tablas de hechos
- **etl/**: Scripts de procesos ETL (Extract, Transform, Load)
- **views/**: Vistas para consultas y reporting

## Convenciones

- Prefijo `Dim` para tablas de dimensiones
- Prefijo `Fact` para tablas de hechos
