# AdventureWorksDW2022MDAGR

## DataWareHouse Adventure Works

Este proyecto implementa un Data Warehouse basado en la base de datos AdventureWorksDW2022 de Microsoft, diseñado para análisis empresarial y reporting.

### Descripción del Proyecto

Adventure Works es una base de datos de ejemplo de Microsoft que representa una empresa ficticia de fabricación y venta de bicicletas. Este Data Warehouse está diseñado para:

- Análisis multidimensional de ventas
- Reporting financiero
- Análisis de clientes y comportamiento de compra
- Forecasting y predicción de tendencias
- Análisis de inventario y productos

### Estructura del Data Warehouse

El proyecto incluye:
- **Dimensiones**: Estructuras para análisis de datos por diferentes perspectivas (tiempo, geografía, productos, clientes)
- **Tablas de Hechos**: Métricas y medidas cuantificables del negocio
- **ETL**: Procesos de extracción, transformación y carga de datos
- **Cubos OLAP**: Estructuras multidimensionales para análisis rápido

### Tecnologías

- Microsoft SQL Server
- SQL Server Analysis Services (SSAS)
- SQL Server Integration Services (SSIS)
- SQL Server Reporting Services (SSRS)
